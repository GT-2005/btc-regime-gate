# Docs

Put the technical report PDF here, plus anything else worth keeping:

- `LATENT_Technical_Report.pdf` — the full methodology
- `folds.csv` — fold-by-fold results, once you export them
- notes on parameter choices and what you tried that didn't work

The last one is worth more than it sounds. Six months from now you will not
remember why you set `signal_buy_thresh` to 0.62 rather than 0.60.
